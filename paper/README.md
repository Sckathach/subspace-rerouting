## Install
This paper requires Typst v0.12.0. You can install it using <https://github.com/typst-community/typst-install>: 
```shell
curl -fsSL https://typst.community/typst-install/install.sh | sh -s 0.12.0
```

## Use
Compile on save:  
```shell
typst watch paper/main.typ
```
